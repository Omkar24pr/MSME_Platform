# Website Layout Modification Prompt



Modify this modern, responsive, professional MSME support platform website with a clean UI using a white, blue, and light-gray color scheme. 



## 1. Landing Page / Main Dashboard



### Header Section



* Add a horizontal hero/banner image at the top of the page.

* The banner should span the full width of the screen.

* Keep the height moderate (approximately 250–350px) so it does not occupy too much vertical space.

* Add a short welcome message overlaying the banner image.



### Navigation Bar



Place the navigation bar directly below the header banner.



#### Center Navigation Items



* About Us

* Services

* Articles

* Notice Board



#### Right Side Navigation Items



* Login

* Register

* Demo



The navbar should remain fixed/sticky while scrolling.



---



## 2. Services Section



Create a section titled **"Our Services"**.



Display 5 clickable service cards in a responsive grid layout.



### Service Cards



1. Mentorship

2. Training

3. Business Consultancy

4. Marketing Support

5. Technology Support



Each card should contain:



* Relevant icon

* Service title

* Short description

* Hover animation

* Clickable button



### Mentorship Service Navigation



When a user clicks the **Mentorship** card, redirect them to a dedicated Mentorship page.



---



## 3. Mentorship Page



Create a separate page for mentors.



### Mentor Listing



Display mentors using cards.



Each mentor card should contain:



* Profile photo/avatar

* Mentor name

* Area of specialization

* Short bio

* Years of experience

* "View Profile" button



Example specializations:



* Business Strategy

* Finance

* Marketing

* Technology

* Startup Growth

* Operations



All mentor cards should be clickable and open a detailed mentor profile page.



---



## 4. Articles Section



On the homepage display only the latest 3 article cards.



Each article card should contain:



* Featured image

* Article title

* Publication date

* Short summary (2–3 lines)

* Read More button



### More Articles Button



Place a **"More Articles"** button in the top-right corner of the Articles section.



When clicked, navigate to a dedicated Articles page.
